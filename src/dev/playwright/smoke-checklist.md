# Playwright Smoke Checklist

1. Open app on `http://localhost:3000`.
2. Import image using picker.
3. Draw rectangle and click `Apply`.
4. Draw ellipse and click `Apply`.
5. Draw line and click `Apply`.
6. Draw freehand and click `Apply`.
7. Use `Ctrl/Cmd+Z` and `Ctrl/Cmd+Y`.
8. Select object, move it, resize it.
9. Copy with `Ctrl/Cmd+C` and paste with `Ctrl/Cmd+V`.
10. Export PNG and verify file is downloaded.
